Voice Digitizer

Designed a voice digitizer using an 8086 microprocessor chip and peripheral devices. The system takes in an audio/voice signal ranging from 0-5V from a microphone. This output is fed into an 8-bit A2D converter sampled at 1000Hz. After the audio sequence has been digitized, it is stored in RAM. The stored audio signal can then be reproduced with a custom delay which entered using a keypad with values ranging from 0-9. A seven-segment display is used to display this value. The stored audio is passed through an 8-bit D2A converter and can be reproduced using a audio output device such as a speaker.

We designed a simulation of this using Proteas and used emu8086 to write the Assembly Level Code to run the simulation.


Some of the key components used for designing the circuit are:
- 8086 Microprocessor
- 8255 Programmable Peripheral Interface
- 8259 Programmable Interrupt Controller
- 8254 Programmable Interval Timer
- 8284 Clock Generator
- 6116 2K RAM
- 2716 2K ROM
- ADC0808 Analog to Digital Converter
- DAC0808 Digital to Analog Converter
- 74244 Octal Buffers
