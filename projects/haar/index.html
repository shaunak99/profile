Object Detection & Blurring Using Haar Cascades For Privacy Protection

OpenCV provides a wide set of libraries for various computer vision and image processing related tasks. Using OpenCV's APIs, a simple tool for Privacy Protection was created that detects number plates in dash cam footage and blurs them.

To detect the number plate in the footage, we used Haar cascade classifiers. Edge and Line Haar features were used to detect the Region of Interest. Having obtained this, we use a simple median blur to mask the number plate details and thus protect the privacy of the individual.
